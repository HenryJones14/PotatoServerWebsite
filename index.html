What is this website for?

  I made this website so I could learn HTML and website design. It is also a place where I can store a coherent list of changes I would make to “fix” Fortnite Competitive. This whole list of changes is only my opinion! Please don't get offended by it just because you like to w-key.

  All of the changes are thought through as if Fortnite would be a survival PVP game. That means that the changes will award consistent players that can outlive others. Some other changes are made so it would be harder for “bush campers” to outplace the better players.
