About

 This project was made for me to learn website design and for me to make a coherent list of changes I would make to “fix” Fortnite Competitive. This whole list of changes is only my opinion! Please don't get offended by it just because you like to w-key.

 All of the changes are thought out as if Fortnite would be a survival of the fittest game. That means that the changes will award consistent players that can outlive other players. Some other changes are made so “bush campers” don't outplace the better players.
